# Enterprise Standards

This memory pack contains canonical enterprise standards for AI assistants and development teams.

## Purpose

Provides authoritative guidance on approved patterns, tools, and practices across the organization.

## Structure

Standards are organized by domain:
- `enterprise/terraform/` — Infrastructure as Code standards
- `enterprise/ado/` — Azure DevOps pipeline standards
- `enterprise/security/` — Security and compliance standards
